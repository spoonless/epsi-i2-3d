Idées de projet¶

Visualisation des tremblements de Terre

L’API Web de USGS fournit des informations sur les tremblements de Terre (localisation, magnitude…). Pour obtenir la liste des tremblements de Terre pour les 30 derniers jours :

https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/significant_month.geojson

Grâce aux résultats fournis par cette API:

  • affichez la position de chaque tremblement de Terre sur une sphère représentant la Terre.

  • un tremblement de Terre peut être représenté par une cercle dont la taille et la couleur dépend de son intensité

  • un petit panneau fournit les infos chiffrés sur l’événement : date, intensité…

  • Si l’intensité d’un tremblement de Terre dépasse un certain seuil (défini par l’utilisateur), ajoutez un élément graphique pour attirer l’attention de l’utilisateur et déplacez la caméra pour voir l’événement

Note

Pour réaliser ce projet, une tâche importante consiste à positionner correctement un point sur le globe selon sa longitude et sa latitude.

Pour localiser un tremblement de Terre, vous pouvez peindre directement une texture en utilisant une CanvasTexture (vous trouverez un exemple ici)

Visualisation de données par pays

Le site REST Countries fournit une API Web pour collecter des données sur des pays.

Par exemple, des informations sur la France :

https://restcountries.eu/rest/v2/alpha/fr

On peut obtenir des informations à partir d’un code pays en plaçant à la fin de l’URI le code du pays.

Affichez les données sous la forme de bâtons qui sont positionnés à la verticale de chaque pays sur une sphère représentant la Terre.

L’utilisateur peut choisir les statistiques qui l’intéressent :

  • population

  • coefficient de Gini

Un panneau s’affiche pour chaque pays pour donner les informations chiffrées.

L’utilisateur peut sélectionner les pays qui l’intéressent en cliquant directement sur le globe terrestre.

Pour ce mini-projet, limitez-vous à quatre ou cinq pays.

Note

Utilisez une texture pour la color map qui affiche les frontières entre pays

Pour afficher les bâtons vous devrez au préalable avoir choisi un point sur la sphère pour chaque pays supporté.

Pour permettre de sélectionner un pays, pensez que le Raycaster peut fournir les coordonnées UV du point touché par le rayon. On peut ainsi avoir une map de pays dans laquelle chaque pays est associé à une couleur.

diaporama 3D

Fournissez une bibliothèque JavaScript pour transformer une portion de page HTML en diaporama 3D. Vous devez créer une fonction JavaScript :

var domElement = slideshow3d(source);
source

L’élément DOM contenant la liste des éléments du diaporama. Cet élément peut contenir des éléments <p> pour le texte et <img> pour les images du diaporama.

<div id="source">
  <p>Bonjour</p>
  <img src="image.png"/>
  <p>Merci</p>
</div>
domElement

La fonction retourne l’élément canvas qui permet l’affichage 3D.

L’utilisateur peut passer d’un slide au suivant en appuyant sur une touche ou en cliquant. Prévoyez plusieurs transitions 3D entre les slides (déplacement de caméra, rotation des slides…).

Idéalement, les transitions pourraient être données par un attribut data-transition sur chaque élément HTML.

<div id="source">
  <p data-transition="left-rotate">Bonjour</p>
  <img data-transition="zoom-in" src="image.png">
  <p data-transition="right-rotate">Merci</p>
</div>

Note

Pour le texte 3D, vous pouvez utiliser la classe TextBufferGeometry.
